# Mesto. Проектная работа для Яндекс.Практикум

Учебный проект, интерактивный сервис для добавления фотографий.

В ходе выполнения работы рассмотрены следующие темы:

* Адаптивная верстка;
* Grid Layout;
* Медиа-запросы.
* DOM

## Стек технологий

* HTML
* CSS
* JS
